# FishTank
**FishTank** 는 **2D 힐링 경영 시뮬레이션** 입니다.

자신만의 작은 바다를 가꾸세요 !

플레이어는 물고기들과 교감 ~~(배설물 치우기)~~ 을 통해 힐링할 수 있습니다 !

다양한 물고기와 장식품을 수집하고 쾌적한 환경을 유지하면서 작은 바다를 만들어 봐요 !

<br>

## 조작
- **마우스 좌클릭 :** 상호작용
- **마우스 우클릭, Enter :**  도구 탭 열기/닫기
- **Space :** 상점 탭 열기/닫기
- **휠 스크롤 :** 도구 변경
- **Esc :** 설정 창 열기/닫기

상점탭과 도구탭은 마우스 클릭으로도 열고 닫을 수 있습니다 !

<br>

## 살아있는 물고기
**FSM** 기반의 AI로 구현된 물고기들이 배고픔을 느끼고 먹이를 찾아다닙니다.

물고기들에게 먹이를 주고 수조 청소를 하며 물고기의 성장을 도와주세요.

<br>

## 저장
JSON 파일로 플레이어의 데이터를 저장해 언제든 다시 불러와 플레이할 수 있습니다.
